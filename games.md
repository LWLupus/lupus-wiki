## Games (DDL)

* [🌟**AnkerGames**](https://ankergames.net/) - New ui/ux less frequently uploads games.
* [🌟**SteamRIP**](https://steamrip.com/) - Old ui/ux but frequently uploads games.
* [🌟**Virgil Game Search**](https://virgil.samidy.com/Games/) - Safe game searcher.

## Game Launcher/downloader

* [🌟**Hydra**](https://hydralauncher.gg/)/[Downloader Plugins](https://library.hydra.wiki/sources)/[Themes](https://hydrathemes.shop/) - Great launcher. Love the UI. It lacks on some features but overall the best one. Almost all download sites in one launcher.
* [**Ascendara**](https://ascendara.app/) - The UI feels like material ui so smooth but only downloads from **SteamRIP**
